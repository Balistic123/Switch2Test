# Switch2Test

1. copy and paste or download the scripts and html file on this repo and place them in the directory
2. make sure both your switch 2 and pc/laptop are on the same wifi (ubuntu or windows work)
3. do ipconfig or ifconfig depending on what operating system your on and mark it down somewhere.
4. modify the lol.py with the directory on where your html file is.
5. run lol.py
6. go to your switch 2 and select the same network your pc/laptop is on.
7. click change network settings and scroll down to where proxy settings is
8. turn it on then change the ip to your laptop/pc ip
9. click save and connect to the wifi
10. it should say this network requires registration.
11. if it works it should bring you to the html page.
